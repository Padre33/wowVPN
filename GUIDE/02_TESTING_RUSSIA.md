# ShadeVPN — Полный гайд тестирования в России

## Что проверять

### Обязательные проверки (базовые)

#### 1. Проверка IP адреса
- Зайти: **https://2ip.ru**
- Должен показать: IP = `185.204.52.135`, страна = Нидерланды
- Если показывает российский IP → VPN НЕ работает

#### 2. DNS Leak Test (утечка DNS)
- Зайти: **https://www.dnsleaktest.com**
- Нажать "Extended test"
- Все DNS серверы должны быть НЕ из России
- Если видишь российский DNS → утечка, небезопасно

#### 3. WebRTC Leak Test
- Зайти: **https://browserleaks.com/webrtc**
- Не должен показывать настоящий IP
- Если показывает два IP (VPN + настоящий) → утечка через WebRTC

#### 4. Проверка скорости
- Зайти: **https://fast.com** (сервис Netflix)
- ИЛИ: **https://www.speedtest.net**
- Нормальная скорость: 30-80 Mbps
- Если менее 5 Mbps → проблема с сервером

### На каких сетях проверять

#### WiFi (домашний):
```
1. Подключиться к домашнему WiFi
2. Включить VPN
3. Проверить 2ip.ru → должен быть IP VPN сервера
4. Открыть linkedin.com → должен открыться
5. Позвонить в Discord → должно работать
```

#### Мобильный интернет (4G/5G):
```
1. Отключить WiFi, остаться на мобильном
2. Включить VPN
3. Проверить 2ip.ru → должен быть IP VPN сервера
4. Открыть linkedin.com → должен открыться
5. Проверить скорость на fast.com
```

#### Публичный WiFi (кафе/аэропорт):
```
1. Подключиться к публичному WiFi
2. Включить VPN
3. Проверить 2ip.ru
4. Это самый важный тест — публичные сети часто блокируют VPN
```

### Что делать если НЕ работает

#### VPN не подключается:
```
1. Проверить интернет без VPN (работает?)
2. Попробовать переключиться WiFi ↔ мобильный
3. Перезапустить приложение
4. Если не помогает → порт 443 может быть заблокирован провайдером
```

#### Подключился но сайты не открываются:
```
1. Проверить 2ip.ru (IP сменился?)
2. Если IP сменился но сайты не грузятся → проблема DNS
3. Вручную поставить DNS: 1.1.1.1 или 8.8.8.8
4. Перезапустить VPN
```

#### Медленный интернет через VPN:
```
1. Нормально: скорость падает на 10-30%
2. Если падает на 70%+ → проблема с сервером
3. Проверить: не перегружен ли сервер (много клиентов)
4. Проверить: пинг до сервера (ping 185.204.52.135)
   - Менее 100ms — хорошо
   - 100-200ms — терпимо
   - Более 200ms — нужен сервер ближе к РФ (Финляндия)
```
