# Руководство по запуску Админ-Панели (ShadeVPN)

Для того чтобы наша красивая панель управления работала, она поделена на две части: **Frontend** (лицо/дизайн) и **Backend** (мозг/Python).

Их нужно запускать в терминале (командной строке / PowerShell) одновременно. Открой **два разных терминала** в VS Code или Windows.

---

### Терминал №1: Запуск Frontend (Дизайн из Figma)

В первом терминале мы запускаем сервер Vite, который будет крутить React-дизайн.

1. Перейди в папку с дизайном:
```powershell
cd "C:\Users\user\Desktop\contract checker\wowVPN_repo\figmadesighn"
```

2. Запусти сервер разработчика:
```powershell
npm run dev
```

Она напишет что-то вроде:
`➜  Local:   http://localhost:5173/`
👉 **Кликни по этой ссылке (с зажатым Ctrl), чтобы открыть панель в браузере!**

---

### Терминал №2: Запуск Backend (Python API)

Во втором терминале мы запускаем сервер FastAPI, который будет связывать панель с нашими реальными ключами и `clients.json`.

1. Перейди в папку с бэкендом:
```powershell
cd "C:\Users\user\Desktop\contract checker\wowVPN_repo\shadevpn-admin-backend"
```

2. Активируй вируальное окружение (venv) и запусти сервер:
```powershell
.\venv\Scripts\activate
uvicorn main:app --reload --port 8443
```

Она напишет `Uvicorn running on http://127.0.0.1:8443`. Бэкенд готов!

---

### Как это будет работать в будущем на сервере?
Когда мы всё закончим и протестируем на твоём компьютере, я соберу Frontend в один статический пакет (одна папка с файлами), и Python (FastAPI) будет сам раздавать этот дизайн. То есть на продакшене (VPS) нам понадобится **всего одна команда** для запуска всей логики VPN и панели!
